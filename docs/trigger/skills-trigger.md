# Skills Trigger Guide

## `/architecture-patterns`

패키지 구조, 도메인 설계, 헥사고날 아키텍처, 계층 의존성, 새 도메인 생성

## `/clean-code`

Self-validating entity, Tell Don't Ask, 팩토리 메서드, 트랜잭션 전략, 도메인 모델 설계, 검증 로직

## `/tdd-workflow`

기능 구현, 테스트 작성, 버그 수정, Cucumber 시나리오, 새 기능 요청

## `/create-pull-request`

PR 생성, 풀 리퀘스트, 변경사항 제출, 머지 요청

## `/code-review-excellence`

코드 리뷰, PR 리뷰, 코드 품질 피드백

## `/debugging-strategies`

버그, 에러, 디버깅, 테스트 실패, 성능 이슈, 스택 트레이스 분석

## `/skill-creator`

새 스킬 생성, 스킬 수정, 스킬 작성 가이드
