# 용어 사전 (Glossary)

> 이 문서는 **빠르게 용어를 찾아 확인하는 참고서**입니다.
> 기획자, PM, 프론트엔드, 백엔드 모두가 같은 뜻으로 용어를 이해하기 위한 것입니다.

---

## Match Context (경기 관련)

### 경기 (Match)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 경기 | Match | 특정 날짜, 시간, 장소에서 하는 농구 경기 |
| 주최자 | Host | 경기를 만들고 관리하는 사람 |
| 참여자 | Participant | 경기에 확정된 사람 (최대 n명) |
| 코트 | Court | 경기를 할 농구 코트 |
| 시간대 | Time Slot | 경기를 할 시간대 (예: 내일 오후 7시~9시) |
| 최대 참여자 | Max Participants | 한 경기에 참여할 수 있는 최대 인원 |
| 현재 참여자 수 | Current Participant Count | 현재 경기에 신청/확정된 참여자 수 |

### 경기 상태 (Match Status)

| 상태명 | 영문명 | 설명 |
|--------|--------|------|
| 모집 중 | RECRUITING | 인원이 부족해서 아직 신청을 받는 중 |
| 확정됨 | CONFIRMED | 최대 인원이 모두 모인 경기 |
| 완료됨 | COMPLETED | 경기 시간이 지나 끝난 상태 |
| 취소됨 | CANCELLED | Host가 경기를 취소한 상태 |

---

## Participation Context (신청 관련)

### 신청 (Participation)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 신청 | Participation | 사용자가 어떤 경기에 참가하고 싶다는 의사 표현 |
| 신청자 | Applicant | 특정 경기에 신청한 사용자 |
| 신청 대상 | Target | Applicant가 신청한 Match (경기) |
| 신청 시간 | Applied At | 신청을 한 시각 |

### 신청 상태 (Participation Status)

| 상태명 | 영문명 | 설명 |
|--------|--------|------|
| 승인 대기 | PENDING_APPROVAL | 경기에 신청했지만 Host 승인 대기 중 |
| 승인됨 | APPROVED | Host가 승인했지만 경기 확정 전 |
| 거부됨 | REJECTED | Host가 신청을 거부함 |
| 확정됨 | CONFIRMED | Host가 n명 승인 완료 후 자동 확정됨 |
| 완료됨 | COMPLETED | 경기가 정상적으로 끝남 |
| 노쇼 | NO_SHOW | 확정된 경기에 나타나지 않음 |
| 취소됨 | CANCELLED | 신청 취소 또는 경기 자체 취소됨 |

---

## User Context (사용자 관련)

### 사용자 (User)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 사용자 | User | 서비스에 가입한 개인 |
| 프로필 | Profile | 사용자의 기본 정보 (이름, 전화, 이메일, 사진) |
| 이름 | Name | 사용자의 이름 |
| 전화번호 | Phone | 사용자의 전화번호 |
| 이메일 | Email | 사용자의 이메일 주소 |
| 프로필 사진 | Profile Image | 사용자의 프로필 사진 |

### 신뢰도 (Reputation)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 신뢰도 | Reputation | 다른 사람들이 평가한 사용자의 신뢰성 점수 |
| 신뢰도 점수 | Reputation Score | 평점들의 평균 (1.0 ~ 5.0) |
| 평점 | Rating | 경기 후 상대방이 준 별점 (1~5점) |
| 리뷰 | Review | 경기 후 상대방이 쓴 평가 의견 |
| 노쇼 기록 | No-Show Record | 확정된 경기에 나타나지 않은 기록 |
| 노쇼 횟수 | No-Show Count | 누적 노쇼 횟수 |

### 노쇼 (No-Show)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 노쇼 | No-Show | 확정된 경기에 나타나지 않은 것 |

## Location Context (장소 관련)

### 코트 (Court)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 코트 | Court | 농구를 할 수 있는 실제 장소 |
| 코트 이름 | Name | 코트의 이름 (예: "광주 시청 농구장") |
| 위치 | Location | 지도상의 정확한 위치 (위도, 경도) |
| 위도 | Latitude | 위도 좌표 |
| 경도 | Longitude | 경도 좌표 |
| 시설 정보 | Facility Info | 실내/실외, 조명 여부, 주차장, 탈의실 등 |
| 실내/실외 | Indoor/Outdoor | 코트가 실내인지 실외인지 여부 |
| 조명 | Lighting | 야간 경기 가능 여부 |
| 주차장 | Parking | 주차장 있는지 여부 |
| 탈의실 | Changing Room | 탈의실 있는지 여부 |

### 위치 기반 검색 (Location-based Search)

| 한글명 | 영문명 | 설명 |
|--------|--------|------|
| 사용자 위치 | User Location | 사용자의 현재 GPS 위치 (위도, 경도) |
| 검색 반경 | Search Radius | 경기 검색 범위 (기본 2km) |
| 거리 | Distance | 사용자 위치와 코트 간 직선 거리 |
| 지도 마커 | Map Marker | 지도에 표시되는 경기 위치 표시 |
| 근처 경기 | Nearby Matches | 사용자 반경 내의 경기 목록 |

## Context별 용어 빠른 참조

### Match Context
경기, 주최자, 참여자, 코트, 시간대, 최대 참여자, 모집 중, 확정됨, 완료됨, 취소됨

### Participation Context
신청, 신청자, 신청 대상, 승인 대기, 승인됨, 거부됨, 확정됨, 완료됨, 노쇼, 취소됨

### User Context
사용자, 프로필, 신뢰도, 신뢰도 점수, 평점, 리뷰, 노쇼 기록

### Location Context
코트, 위치, 위도, 경도, 시설 정보, 사용자 위치, 검색 반경, 거리, 지도 마커, 근처 경기
